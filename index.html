Remote Laboratory

What is Remote Laboratory?

  • Python, HTML5, and Javascript programming languages are mixed to create a set of a new type of simulators. The simulators are ... normal labs performing in laboratories. The new type of simulators are called Remote Laboratory".
  • Unlike typical simulators that are designed to provide information only, each remote laboratory provides a simulating apparatus with which students are required to perform a set of measurements, analysis, and determination for a number of projects.
  • Instructors are provided, upon request, a small Python program and a Python pickle file that contains remote laboratories.
  • Instructors use the program to generate a set of HTML files, each of which is assigned to a student with a student identification number and contains all the information of layout of the lab and the projects.
